🏢 Multi-tenancy — Slik fungerer det

FieldFlow v2 er bygget for å serve flere kunder samtidig fra én og samme database. Dette kalles multi-tenancy, og er kjernen i en SaaS-plattform.

Innloggingsflyt med Auth0 Organizations:
Bruker logger inn
→
Auth0 utsteder JWT-token med org_id
→
FastAPI mapper org_id → tenant_id
→
PostgreSQL RLS filtrerer data per tenant
→
Bruker ser kun sine data
💡 Enkelt forklart: Hedengren og en annen installatørbedrift kan begge bruke FieldFlow v2 samtidig. De deler samme database, men PostgreSQL sine Row Level Security-regler sørger for at ingen kan se hverandres data — akkurat som at to leiligheter i samme bygård er låst.

🛠️ Teknologi-stakken

Backend
FastAPI
Moderne Python web-rammeverk. Async, raskt, auto-genererer API-dokumentasjon.
Database ORM
SQLAlchemy 2.0
Kobler Python-kode til PostgreSQL. Async-støtte med asyncpg.
Database
PostgreSQL
Supabase i cloud, lokal Docker-instans for utvikling.
Migrasjoner
Alembic
Versjonshåndtering av databaseskjema. Som Git, men for tabeller.
Autentisering
Auth0 Organizations
B2B-innlogging. Hver kunde er én Organization med egne brukere.
Frontend
HTMX + Tailwind
Dynamiske sider uten tung JavaScript. Alpine.js for modaler og interaktivitet.
Deploy
Railway
Automatisk deploy ved push til GitHub main-branch.
Lokalt
Docker + WSL2
Kjører samme miljø lokalt som i cloud. Ubuntu via WSL2 på Windows.
Testing
pytest + asyncio
114 automatiserte tester. Kjøres i minne med SQLite, ingen ekstern DB.

📐 Kodestruktur — lagdelt arkitektur

Koden er organisert i lag, der hvert lag har ett ansvar. Dette gjør det enkelt å endre én ting uten å ødelegge noe annet.

🌐
Routers
Mottar HTTP-forespørsler (GET, POST, PUT, DELETE). Validerer input med Pydantic-skjema.
app/routers/
⚙️
Services
Forretningslogikk. F.eks. "kan ikke slette en region som har teknikere".
app/services/
📦
Repositories
All kommunikasjon med databasen. Søk, paginering, filtrering.
app/repositories/
🗂️
Models
Databasetabellene som Python-klasser. Tenant, User, Region, Technician, Job osv.
app/models/
🔒
Dependencies
Autentisering og RBAC. Kjøres automatisk på alle beskyttede endepunkter.
app/dependencies.py
